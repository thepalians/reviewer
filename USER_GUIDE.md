## üîß Troubleshooting

### Common Issues

#### Login Problems

**Issue**: Can't log in
**Solutions**:
1. Check email/password are correct
2. Clear browser cache and cookies
3. Try password reset
4. Disable browser extensions
5. Try different browser
6. Check 2FA code is current (refresh)
7. Contact support if issue persists

**Issue**: Forgot password
**Solution**:
1. Click "Forgot Password"
2. Enter email address
3. Check email for reset link
4. Create new password
5. Log in with new password

#### Task Issues

**Issue**: Can't see tasks
**Solutions**:
1. Check your tier's daily limit
2. Verify KYC is completed
3. Check account status (not suspended)
4. Refresh page
5. Clear cache
6. Try different browser

**Issue**: Task submission failed
**Solutions**:
1. Check file size (max 5MB per image)
2. Ensure all required fields filled
3. Verify internet connection
4. Try again after few minutes
5. Contact seller for clarification

**Issue**: Task rejected
**Reasons**:
- Poor quality review
- Missing required keywords
- Fake/invalid screenshot
- Plagiarized content
- Not following instructions

**Action**: Review rejection reason and improve next time

#### Payment Issues

**Issue**: Payment failed
**Solutions**:
1. Check bank balance
2. Verify card details
3. Try different payment method
4. Check payment gateway status
5. Contact bank
6. Try smaller amount
7. Contact support

**Issue**: Withdrawal pending
**Timeline**: 
- Processing: Up to 48 hours
- Holidays/weekends may delay
- Bank processing: 1-2 business days

**Action**: Wait for processing. Contact support if >72 hours.

**Issue**: Wrong amount received
**Action**:
1. Check transaction details
2. Verify commission calculation
3. Check deductions (fees, taxes)
4. Submit support ticket with proof

#### Quality Score Issues

**Issue**: Low quality score
**Improve by**:
1. Write original reviews (no plagiarism)
2. Use proper grammar and spelling
3. Include required keywords naturally
4. Follow all instructions carefully
5. Upload genuine screenshots
6. Be detailed and helpful
7. Maintain consistent quality

#### KYC Issues

**Issue**: KYC rejected
**Common reasons**:
- Blurry documents
- Document expired
- Name mismatch
- Incomplete information
- Invalid document type

**Solution**: Re-upload with correct documents

**Issue**: KYC pending too long
**Action**: If >48 hours, contact support

### Error Messages

#### "Session expired"
- **Cause**: Inactive for 30+ minutes
- **Solution**: Log in again

#### "Insufficient balance"
- **Cause**: Wallet balance too low
- **Solution**: Add money to wallet (sellers) or complete more tasks (reviewers)

#### "Daily limit reached"
- **Cause**: Completed maximum tasks for your tier
- **Solution**: Wait for next day or upgrade tier

#### "Account suspended"
- **Cause**: Violation of terms or suspicious activity
- **Solution**: Contact support for details

#### "Payment gateway error"
- **Cause**: Gateway down or configuration issue
- **Solution**: Try different gateway or contact support

#### "File too large"
- **Cause**: Image >5MB
- **Solution**: Compress image and re-upload
